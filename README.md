# UmiBird.io

A browser-based Flappy Bird-style game built using JavaScript, HTML, and CSS.

## Overview

This project is a simple interactive web game inspired by Flappy Bird. It demonstrates core front-end development concepts including DOM manipulation, animation, and event handling.

The player controls a character that must navigate through obstacles while avoiding collisions.

## Features

* Real-time game loop and animation
* Collision detection for obstacles
* Keyboard input handling for player control
* Score tracking system
* Responsive visual updates using CSS and JavaScript

## Tech Stack

* JavaScript
* HTML
* CSS

## Project Structure

```
.
├── index.html
├── flappybird.js
├── flappybird.css
├── *.png (game assets)
```

## How It Works

1. The game loop continuously updates player position and obstacle movement
2. User input (keyboard) controls vertical movement
3. Collision detection checks for contact with pipes or ground
4. Score increases as the player successfully passes obstacles
5. Game resets upon collision

## How to Run

1. Open `index.html` in a web browser
2. Press a key to start the game
3. Use the keyboard to control the character

## Purpose

This project was built to practice front-end development concepts such as animation, event handling, and interactive UI behavior.
